# Mobile Data Capture and Web Sync Prototype

This repository contains a prototype system for offline mobile data capture and cloud synchronization.

## Main Components

- Flutter Android mobile app
- SQLite local mobile database
- React + Tailwind CSS web application
- Node.js + Express.js backend API
- PostgreSQL database hosted on Supabase
- Supabase Storage for image uploads
- Public cloud deployment using Vercel, Render, and Supabase
